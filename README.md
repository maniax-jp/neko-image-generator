# 猫画像生成器 (Neko Image Generator)

Puter.js を使用して、入力したプロンプトに基づいた可愛い猫の画像を生成するシンプルなウェブアプリケーションです。

## 🚀 デモ
以下のURLで動作を確認できます：
[https://maniax-jp.github.io/neko-image-generator/](https://maniax-jp.github.io/neko-image-generator/)

## ✨ 特徴
- **AI画像生成**: Puter.js の AI API を利用して、テキストから画像を生成します。
- **シンプル設計**: Vite を使用した高速な開発環境とモダンなフロントエンド構成。
- **自動デプロイ**: GitHub Actions により、`main` ブランチへのプッシュ時に自動的に GitHub Pages へ公開されます。

## 🛠️ 技術スタック
- **Frontend**: HTML, CSS, JavaScript
- **Build Tool**: [Vite](https://vitejs.dev/)
- **AI API**: [Puter.js](https://puter.com/)

## 💻 ローカルでの実行方法

### 1. リポジトリのクローン
```bash
git clone https://github.com/maniax-jp/neko-image-generator.git
cd neko-image-generator
```

### 2. 依存関係のインストール
```bash
npm install
```

### 3. 開発サーバーの起動
```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスして動作を確認してください。
